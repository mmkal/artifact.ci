# artifact.ci

artifact.ci is splitting into three focused surfaces:

- docs on Astro + Starlight
- signed-in app UI on TanStack Start + Better Auth
- binary delivery on a Cloudflare frontdoor Worker

That split keeps markdown docs simple, keeps product UI in React, and lets the thing that actually matters most - serving artifacts quickly and correctly - stay at the edge.
